Class: info:ComputerFile

"A computer file is a block of arbitrary information, or resource for storing information, which is available to a computer program and is usually based on some kind of durable storage. A file is durable in the sense that it remains available for programs to use after the current program has finished. It is indexed in a computer file system and has a file path within." [lang: en]

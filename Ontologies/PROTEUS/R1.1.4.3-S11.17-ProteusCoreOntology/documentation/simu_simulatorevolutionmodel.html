Class: simu:SimulatorEvolutionModel

"It is an evolution model that allows to provide one timeframe to one system. it can be synchronised with other simulators. The system manipulated by the simulator does not need to be specific. It can be an hardware (reduced scale models), a software (HLA like) or an hybrid one where there are software and hardware components (hybrid simulator). It corresponds mainly to the simulation theory that is used." [lang: en]

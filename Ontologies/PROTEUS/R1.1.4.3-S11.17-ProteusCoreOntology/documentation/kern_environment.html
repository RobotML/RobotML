Class: kern:Environment

"This environment groups everything that exist. Nothing is outside and everything lives in it. An alias for environment could be Universe or World in an ontologic meaning. This is the highest possible "container" for the physical objects to exist. Considering the scenario to be represented using this concept it expresses also the boundary we imagine for the problem. As an example, for a mobile on ground robot there is no need to represent anything concerning water. The difference between physical objects and THE environment is that the PhysicalInteraction are part only of the latter and propagated into it." [lang: en]

Class: syst:RoboticFunctionalSystem

"This entity defines the functional Architecture that can be found in a Robotic System. The Functional System is the logical view of the Robotic Platform. Thus, we can find function as control, mission management, motion planning, platform management, or security. It can be also seen as acomplete. It is often associated to the software implementing this functional view." [lang: en]

Class: kern:AtomicModel

"This concept groups the different theory that can be used to model one or more system. It is often linnked to a mathematical theory. It also defines the "external" model concept meaning that the model exist, is provided but is only known through its interface. An atomic system can models more than one system. It explains whay the properties models has cardinality more than one. An atomic model beiing an atomic system is thus disjoint from a composite model as it is a composite system." [lang: en]

Class: syst:ActuatorSystem

"Electromechanical System INCLUDING software which to control the effectors, like a mechanical arm driver. IMPORTANT WARNINGS, In the scope of PROTEUS project different comments are added here: - When the "Actuator" word is used alone, it means the Actuator system includes on ly one non decomposable software and one non decomposable hardware. This is of great ambiguity but reflects usage in the community. - There is often a difference between the Motorisation system, able to move the robot or part of it, and the different other systems able to interact with the environment. They can be called Payload systems, Operationan systems, Mission mmodules, etc (as an example, NRBC, Weapons, Communications, ...)" [lang: en]
